# 🤖 AI Educational Chatbot (RAG Project)

This is an AI-powered **Retrieval-Augmented Generation (RAG)** chatbot that allows users to upload PDF documents and ask questions based on the content. The system retrieves relevant information from documents and generates accurate AI responses.

---

## 🚀 Project Overview

This project uses **LangChain + ChromaDB + Streamlit** to build a smart document-based question-answering system.

Users can:
- Upload PDF files 📄
- Process and store document embeddings 🧠
- Ask natural language questions ❓
- Get AI-generated answers based on document content 🤖

---

## 🎯 Features

- 📄 PDF document upload
- 🔍 Semantic search using embeddings
- 🧠 Context-based question answering
- ⚡ Fast retrieval using vector database (ChromaDB)
- 💬 Interactive Streamlit UI
- 🤖 AI-generated responses

---

## 🛠️ Tech Stack

- Python 🐍
- Streamlit 🎈
- LangChain 🔗
- ChromaDB 🗄️
- Hugging Face Embeddings 🤗
